## Glia Widgets SDK

### Project setup
* **Make sure you have** 
	* [Git LFS](https://git-lfs.github.com/)
	* [Xcode](https://developer.apple.com/xcode/)  
	* [CocoaPods](https://cocoapods.org/)
* **Run Terminal commands**
	* `$ cd project-root-path`
	* `$ pod install`
* **Open GliaWidgets.xcworkspace**
